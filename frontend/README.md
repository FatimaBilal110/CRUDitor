# Frontend Login App

A simple user authentication frontend built with **Next.js 14** and **Tailwind CSS**, connected to a backend API (e.g., Node.js/Express + MongoDB).

## 📁 Project Structure

```
src/
├── app/
│   ├── login
│   │      └── page.jsx
│   ├── signup
|   |    └── page.jsx
│   ├── users
│   |     └── page.jsx
```
---
## 🚀 Features

- User Signup & Login
- JWT Authentication
- Protected Routes
- Responsive UI using Tailwind CSS

## 📦 Tech Stack

- Next.js
- Tailwind CSS
- React
- Backend API (Express.js with MongoDB) [in the backend]

## Install Dependency
- npm install
## Strat Development server 
- npm run dev

The app will be running at: http://localhost:3000
